# ionic2-room_view
##Introduction
This project combined [Three.js](https://threejs.org/) and [ionic2](http://ionicframework.com/docs/v2/setup/installation/) to show 360 degree image.
Users can rotate the device and see different angle of the image.

## How to run
Firstly
```
npm install
```
Then
```
ionic serve 

```
Or 
```
ionic run ios
```
Or 
```
ionic run android
```
